 ## 1. 긴급 설비 점검 대상
- 라인 C (제조 라인)
- 1. 컨베이어 벨트 상태 확인
   - 손상 여부, 얼룩 여부, 턱 부러진 부분 여부 등 확인
   - 소요 시간: 1시간
   - 필요 부품: 컨베이어 벨트 검사도구
- 2. 압력 센서 확인
   - 정상적인지 확인, 손상 여부 확인
   - 소요 시간: 30분
   - 필요 도구: 압력 센서 테스터
- 3. 전압 공급 시스템 확인
   - 전압 공급 여부 확인
   - 소요 시간: 1시간
   - 필요 도구: 전압 공급 시스템 테스터

## 2. 공정 파라미터 조정
- 라인 C (제조 라인)
- 현재 압력 설정값 → 권장 압력 설정값: 1000Pa → 1100Pa
- 조정 근거: 금형 온도 설정값 변경으로 인한 불량률 증가 원인 중 하나
- 검증 방법: 압력 센서 값 확인 및 불량률 변화 확인

## 3. 예방 정비 계획
- 일정: 내일 오전 9시
- 대상 설비: 라인 C (제조 라인)
- 정비 내용: 컨베이어 벨트 교체, 압력 센서 교체, 전압 공급 시스템 점검
- 필요 자원: 엔지니어 2명, 기술자 3명, 필요 부품 및 도구

## 4. 기술 개선 프로젝트
- 프로젝트명: 라인별 최적 설정값 추출 및 적용
- 목표: 라인별 불량률 감소
- 필요 기술/도구: 데이터 분석 도구, 통계 분석 도구
- 예상 기간: 1개월
